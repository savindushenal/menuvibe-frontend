# Barista Product Images

This folder should contain high-quality product images for the demo.

## Required Images

1. **cappuccino.jpg** - Classic Cappuccino
2. **brownie.jpg** - Chocolate Brownie  
3. **croissant.jpg** - Butter Croissant
4. **logo.png** - Barista logo (transparent background)

## Image Specifications

- **Format:** JPG for photos, PNG for logo
- **Size:** 800x600px minimum for products
- **Quality:** High resolution, appetizing food photography
- **Style:** Consistent lighting, warm tones

## Sourcing Images

### Option 1: Barista Website/Instagram
- Best for authentic branding
- Screenshot and crop their actual product photos

### Option 2: Free Stock Photos
- **Unsplash:** https://unsplash.com/s/photos/cappuccino
- **Pexels:** https://www.pexels.com/search/coffee/

### Option 3: AI Generated
- Use DALL-E or Midjourney for custom product shots

## Note

For the demo, placeholder emoji icons are used as fallback.
Adding real images will significantly improve the presentation.
